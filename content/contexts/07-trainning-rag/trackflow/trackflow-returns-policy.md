# Returns Policy

TrackFlow's standard reverse logistics process works as follows:

1. The end customer requests a return through the brand's portal.
2. TrackFlow automatically generates the return label if the reason falls
   within the rules configured by the client (e.g. wrong size, defective
   product).
3. The product arrives at the warehouse and an operative inspects and
   classifies it as: fit for resale, fit for reconditioning, or not fit
   (disposal).

Standard return window: 30 days from delivery, unless the client brand has
configured a different window in their contract.

Return costs: the return shipping cost is covered by the client brand,
unless the reason is a TrackFlow error (product damaged in transit or
wrong delivery), in which case TrackFlow covers it.

International returns (purchased in one country, returned in another) are
not covered by the standard automated process and require manual handling
by Sofía Ramos's team (Returns Manager).

TrackFlow does not recondition or resell product on its own account; it
only executes logistics according to the rules each client brand defines.
