# Menu Allergen Guide

Brasaland discloses allergens according to the regulations in effect in
each country (INVIMA in Colombia, FDA in the United States). Floor staff
must be able to answer allergen questions without hesitation, since this is
customer safety information.

Main dishes and their declared allergens:
- Grilled Sirloin (Lomo a la Brasa): gluten-free, dairy-free. Marinade
  contains soy.
- Brasaland BBQ Ribs: contains soy (sauce) and may contain trace peanuts on
  some production lines of the imported sauce.
- Classic Grilled Chicken: gluten-free, dairy-free, nut-free.
- Tropical Salad: contains nuts (cashew) and dairy (feta cheese).
- Corn Arepa (side): contains dairy and egg.
- House Sauce: contains soy and sulfites.

Protocol for customer-reported allergies:
1. The server informs the kitchen before preparation begins.
2. The kitchen uses clean utensils and a surface separate from the general
   prep area.
3. "Zero cross-contamination risk" is never guaranteed — the server must
   communicate this explicitly if the customer has a severe allergy.

There is no certified gluten-free version of the BBQ Ribs due to the
imported sauce; this must always be communicated when asked.
